Web Archive! Las Páginas Que Desaparecieron…

Aug 30th, 2004 7:00 am

Ay! La primera página web que hice… dónde quedó… cony, pero si esta aquí! Alguien me la ha guardado! xD

El servicio de web archive almacena webs al estilo de la caché de google, ¡pero tiene una base de datos de 7 años! Pruébalo tú mismo con tus webs antiguas o demás…

Algunas curiosidades:

Primer theme de gplurv -> aquí
Pagina de microsoft en el 96 -> aquí
Página de kernel.org en el 98 -> aquí
Slashdot en el 97 -> aquí
Barrapunto en el 99 -> aquí
Mi primera web -> aquí i aquí

Mi primera web fué sobre videojuegos, me encataban (y siguen gustándome :P), sobretodo los de consola, y especificando más… la mayoría era de Nintendo. Aún dura on-line con el último aspecto que le di (muy currado, todo tablas… intenta entender el código xDD) y la página tenía bastante éxito en su momento… incluso llegué a salir en la revista Hobby Consolas.

Mis primeras direcciones web (la mayoría ya no existe pero las pongo aquí para recordarlas):

Directas:

http://leo.worldonline.es/joerquep/
http://www.iespana.es/consolas
http://www.iespana.es/lawebdelvideojuego

Redirecciones:

Por aquel entonces yo era un windolin recién conectado a internet, echaba un vistazo al IRC con el Mirc, miraba la web con el mejor navegador sin duda de la época, el Internet Explorer (sí, por aquel entonces el explorer era mejor que el Netscape… de hecho, Netscape nunca me ha hecho mucha gracia, era incómodo para desarrollar y bastante feo antes. Mozilla sí empezó a gustarme a partir de la 1.0 y ahora el Firefox es la cosa más bonita que hay xD)… etc.
La web la empezé con el bloc de notas y al final me quedé con el editor de html homesite, lo consideraba lo mejor de la época.
