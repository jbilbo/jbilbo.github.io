Me parece muy curioso el artículo de [Vidaextra](http://www.vidaextra.com/2007/02/15-nvidia-aun-no-tiene-controladores-para-windows-vista) donde comentan que un modelo de tarjeta gráfica Nvidia no tiene todavía controladores para el nuevo Windows Vista.

Si os fijáis el tono, es el de: “¡con lo cara que es la tarjeta, y
encima no tiene drivers dignos para Windows Vista!” Bien, no digo que
no se deba tener esa actitud ante tal situación… pero cambiemos por un
momento el S.O. y en vez de “Windows Vista” pongamos “Linux”. ¿Qué pasa
ahora? ¿Quién es el malo que no tiene drivers para la tarjeta gráfica?
¿Linux, o la própia Nvidia que no los hace?

Nvidia hace drivers para Linux (al igual que existen unos “beta”
para Vista), pero me parece muy curiosa la comparación… durante mucho
tiempo he oído quejas de este tipo:

- *Uno de los “problemas” que tiene linux es que no tiene drivers para mi (pon aquí tu dispositivo preferido)… ¡maldito linux!*

Y ahora me topo con…

- *Mi (dispositivo preferido) marca X no tiene drivers para Windows Vista… ¡pero qué es esto! ¡maldita compañía X!*

¿Cómo haran para que la gente eche la culpa a la compañía? No
espera… esto sería lo normal. Entonces… ¿Cómo se ha llegado a echar la
culpa a Linux?
